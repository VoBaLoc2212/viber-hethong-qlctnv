# Domain: operations

## Purpose
Capture environment variables, run/deploy paths, and operational dependencies.

## Scope
Included in this domain:
- Runtime configuration variables
- Local and production deployment mechanisms
- CI/CD integration and external services

Excluded from this domain:
- Feature-level business logic design

## Ownership
DevOps / Platform

## Usage
Use this domain for environment setup, release readiness, and production operations.
