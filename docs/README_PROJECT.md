# README (Project Baseline)

## Tóm tắt

Hệ thống Quản lý Ngân sách Chi phí & Thu - Chi Nội bộ, vận hành theo kiến trúc module trong `src/`.

## Tech stack

- Next.js App Router
- PostgreSQL
- Prisma
- Docker + Nginx
- OpenAPI + Zod + generated client
- JWT + RBAC
- S3-compatible storage
- Recharts

## Runtime hiện tại

- Runtime chính: `src/`

## Quickstart

```bash
npm install
cp .env.example .env
docker-compose up -d
```

### Chạy local
```bash
npm run dev
```
